블랙잭 4티어 캐릭터

아래 캐릭터들을 블잭에서 플레이하는 건 권장하지 않습니다.
물론 놀려고 드는 거면 상관없고요.
4티어 캐릭터에는 생존자는 모험가, 도둑, 의사, 주술사, 포워드, 용병, 맹인,
감시자는 광대, 루키노, 봉봉, 하스터, 울보가 있습니다.

4티어 생존자

모험가

책의 스텔스 성능 자체는 좋지만, 스텔스는 칩이 안벌립니다.

도둑

기본 스펙이 나쁜 건 아니지만, 모든 감시자의 기본 스펠은 텔레포트입니다.

의사

블랙잭에서는 진정제가 의미 없습니다. 창판디버프도 존재하고요.
그나마 지렁이일 때 더 빨리 일어난다는 점이 위안이네요.

주술사

주술 디버프도 판이 끝날 때까지 중첩되며, 주술 소지량은 2단계가 최대입니다.
거기다가 치료디버프까지. 주술 자체는 쓸만하지만 그렇다고 써먹을 만한 캐릭은 아닙니다.

포워드/용병

얘네들은 해독속도도 느린데 소지한 아이템이 너무 적습니다.(럭비공 2초, 아대 1개)
어그로도 끌기 힘들고 해독도 느려서 잘 안쓰입니다.

맹인

해독속도는 제일 빠릅니다만, 맹인은 기계공과 달리 직접 해독을 해야 하기에 어그로를 더 많이 끌게 됩니다.
다만, 반향정위 사용 시 감시자의 위치가 라운드 끝날 때까지 보이므로 텔각 보기에는 좋습니다.

4티어 감시자

광대

돌진 2방 맞추는 건 시간이 너무 많이 소모되고, 그렇다고 평타를 맞추자니 뚜벅이인 조커는 9시 생존자를 잡는 데 시간이 많이 소요됩니다.

루키노

노존재감 스킬은 거리좁히기에는 좋지만 심리전 걸기에는 부적합하고, 풀존재감 스킬은 2번 맞춰야 해서 블랙잭에서 사용하기에는 적합하지 않습니다.

봉봉/울보

노존재감 스킬이 매우 약하고, 풀존재감 스킬도 이렇다한 추격상 이점을 제공하지 못합니다.

하스터

촉수도 결국은 스킬판정이고, 풀존재감 스킬도 촉수강화스킬이라서 추격상 이점을 제공하지 못합니다.
